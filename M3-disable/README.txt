This directory contains a SoftConsole v4.x workspace. This SoftConsole workspace
contains Cortex-M3 programs used to configure SmartFusion2 devices before
disabling the SmartFusion2 Cortex-M3. The SmartFusion2 configuration can include
DDR memory controller configuration and/or SERDES configuration used for Ethernet
communications. The configuraion programs are board specific when DDR and/or
SERDES are configured.
